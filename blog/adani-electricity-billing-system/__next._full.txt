## Project Overview

**Client**: PDB Bank  
**Role**: Data Scientist and ML Ops  
**Tech Stack**: AWS EC2, Django, Python, Aadhaar API, PAN API, CIBIL, Equifax, Machine Learning

Revolutionized the loan approval process by building an end-to-end automated system that evaluates creditworthiness in real-time.

## The Problem

Traditional loan processing was a bottleneck:

- **Weeks-long approval times** frustrating customers
- **Manual data entry** prone to human error
- **Limited risk assessment** missing alternative data signals
- **No flexibility** in lending criteria adjustments

## Solution Architecture

### 1. Comprehensive Data Pipeline

Designed system processing **105+ parameters**:

- **KYC Verification**: Real-time Aadhaar and PAN validation
- **Credit Scoring**: CIBIL and Equifax integration
- **Alternative Data**: Income proxies, utility payments, transaction history
- **Fraud Checks**: Cross-verification across multiple data sources

### 2. Dynamic ML Scoring Model

Built intelligent creditworthiness engine:

- **Feature Engineering**: Extracted predictive signals from sparse data -**Alternative Scoring**: Evaluated unscored customers using proxy variables
- **Risk Segmentation**: Classified applicants into risk buckets
- **Explainability**: Provided transparent reasoning for decisions

### 3. Real-Time Admin Panel

Enabled business agility:

- **Dynamic Criteria**: Adjust lending parameters without code deploy
- **A/B Testing**: Experiment with different scoring rules
- **Monitoring Dashboard**: Track approval rates, defaults, portfolio health
- **Audit Trail**: Complete logging for compliance

```python
class LoanScoringEngine:
    def __init__(self, criteria_config):
        self.criteria = criteria_config
        self.models = self.load_ensemble_models()

    def evaluate_applicant(self, applicant_data):
        # Traditional credit score (if available)
        credit_score = self.get_bureau_score(applicant_data)

        # Alternative data scoring for thin-file customers
        if credit_score is None:
            alt_score = self.calculate_alternative_score(
                utility_payments=applicant_data['utilities'],
                income_proxies=applicant_data['income_signals'],
                behavioral_data=applicant_data['transactions']
            )
            return self.apply_criteria(alt_score, 'alternative')

        return self.apply_criteria(credit_score, 'traditional')
```

```python
# Example: Real-time Forex conversion with caching
class ForexConverter:
    def __init__(self, api_key):
        self.api = ForexAPI(api_key)
        self.cache = Redis()

    def convert_to_usd(self, amount, currency, date):
        cache_key = f"forex:{currency}:USD:{date}"
        rate = self.cache.get(cache_key)

        if rate is None:
            rate = self.api.get_historical_rate(
                base=currency,
                target='USD',
                date=date
            )
            self.cache.setex(cache_key, 3600, rate)  # Cache for 1 hour

        return amount * float(rate)
```
