## Project Overview

**Client**: Adani Electricity  
**Role**: Lead Data Scientist  
**Tech Stack**: OpenCV, Django, Vue.js, PWA (Progressive Web Applications), Keras, TensorFlow

Developed an AI-powered system to combat billing fraud by automatically detecting tampered or recaptured meter reading images submitted by customers.

## The Challenge

Utility companies face significant revenue losses from fraudulent meter readings. Traditional manual verification was:

- **Time-consuming**: Required human review of thousands of images
- **Error-prone**: Difficult to spot subtle tampering techniques
- **Scalable issues**: Couldn't handle growing customer base

## Technical Solution

### 1. Frequency-Based Decomposition Pipeline

Implemented advanced image processing to detect **Moiré patterns** - telltale signs of screen-captured images:

- Applied **Fourier Transform** analysis to identify frequency anomalies
- Detected periodic interference patterns invisible to human eye
- Achieved **91% accuracy** in tampering detection

### 2. Multi-CNN Architecture

Built ensemble model combining:

- **OpenCV** for preprocessing and feature extraction
- **PIL** for image manipulation
- **Keras + TensorFlow** for deep learning classification
- Validated with **50+ automated checks** including object detection and boundary frame analysis

### 3. Progressive Web Application

Created seamless customer experience:

- **Vue.js frontend**: Responsive, mobile-first design
- **Django backend**: RESTful API for real-time image processing
- **PWA capabilities**: Offline support, push notifications
- **Real-time validation**: Instant feedback on image quality

## Key Results

- **72% boost in accuracy** compared to previous methods
- **63% faster processing** time
- **29% increase in revenue** through fraud prevention
- **Improved customer experience** with faster, user-friendly billing process

## Project Overview

**Client**: Almavest (International Lending Platform)  
**Role**: Project Leader / System Architect  
**Tech Stack**: AWS Lambda, EC2, CloudWatch, Django, Superset, PostgreSQL, Forex API, Airtable, Google Sheets, SageMaker

Built an automated investment tracking system bridging international borrowers and investors with real-time currency conversion and portfolio health monitoring.

## The Challenge

Almavest faced complex data integration challenges:

- **Diverse data sources**: Google Sheets, Airtable, proprietary DBs
- **International transactions**: Multi-currency with fluctuating exchange rates
- **Legacy systems**: Inconsistent data entry practices
- **Manual tracking**: Excel-based processes error-prone and time-consuming

## System Architecture

### 1. Automated ETL Pipeline (AWS Lambda)

Designed serverless data processing:

- **Multi-source extraction**: Standardized data from 8-10 lending vehicles
- **Loan amortization**: Calculated repayment schedules automatically
- **Status tracking**: Real-time updates on payment status
- **Data validation**: Ensured consistency across disparate sources

### 2. Dynamic Investment Dashboard

Built comprehensive visualization layer:

- **Django backend**: RESTful API serving portfolio data
- **Superset frontend**: Interactive dashboards with drill-down capabilities
- **Real-time Forex**: Dynamic currency conversion using live rates
- **Portfolio health**: Visual indicators of performance vs. expectations

### 3. Intelligent Alert System

Proactive risk management:

- **Deviation detection**: Flagged repayment anomalies
- **Email notifications**: Alerted investors to issues
- **Threshold configuration**: Customizable alert rules
- **Historical tracking**: Trend analysis for early warning

```python
# Example: Real-time Forex conversion with caching
class ForexConverter:
    def __init__(self, api_key):
        self.api = ForexAPI(api_key)
        self.cache = Redis()

    def convert_to_usd(self, amount, currency, date):
        cache_key = f"forex:{currency}:USD:{date}"
        rate = self.cache.get(cache_key)

        if rate is None:
            rate = self.api.get_historical_rate(
                base=currency,
                target='USD',
                date=date
            )
            self.cache.setex(cache_key, 3600, rate)  # Cache for 1 hour

        return amount * float(rate)
```
